Smooth Sailing Properties LLC

Property Management

Smooth Saillng Properties LLC specializes in newer well maintained single family homes for lease. They lease to the general public as well as Clayton Housing Authority and Fulton County Housing Authority Section 8 tenants. They strive to make renting a home a fun and relaxing event. They used to use a generic cookie cutter website on one of the big free hosting sites built for small real estate firms. As they prepare to expand, the needed a more personalized website that would be tailored to their needs. The website is built on modern technologies such as HTML 5, CSS3 and utilizes key Javascript libraries and plugins and is built on the WordPress core utilizing unique real estate plugins. It is also mobile friendly and easy to edit content and property listing all from a simplified admin dashboard.
